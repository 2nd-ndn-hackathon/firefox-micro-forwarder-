Firefox-micro-forwarder

Micro forwarder Firefox add-on

Download this project as a .zip file Download this project as a tar.gz file

Motivation and problem statement

Browser applications may need a local forwarder. A Firefox add-on can provide a micro forwarder common to all browser tabs where users can visit a web page, click to accept a plug-in, and experience an NDN application.

Contribution to NDN

  • Users and developers can experiment with NDN
  • Provide an easy gateway to using NFD

Tasks

  • Review existing work for in-browser functions
  • Implement a FIB/multicast in JavaScript
  • Toolbar to configure and show routes/status
  • Expand NDN-JS PIT as needed for a forwarder
  • Basic content store with simple retention policy

Required knowledge for participants

  • JavaScript programming
  • Forwarder basics (PIT, FIB, CS)
  • Bonus: Experience with Firefox add-ons

Expected outcome

Demonstrate JavaScript ChronChat where two tabs communicate with each other plus a remote user via the in-browser forwarder
